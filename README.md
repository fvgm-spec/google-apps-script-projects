# Google Apps Script Projects

A collection of Google Apps Script mini-projects for automation and data processing.

## Projects

### 1. Bitcoin Price Tracker
- **Description**: Tracks Bitcoin prices and updates Google Sheets
- **Features**: 
  - Real-time price fetching
  - Automated data logging
  - Price alerts and notifications
- **Location**: `/bitcoin-price-tracker/`

## Setup Instructions

1. **Google Apps Script Setup**:
   - Open [Google Apps Script](https://script.google.com/)
   - Create a new project
   - Copy the code from the respective project folder
   - Set up triggers and permissions as needed

2. **Local Development**:
   - Each project has its own folder
   - `.gs` files contain the Google Apps Script code
   - `README.md` in each folder explains project-specific setup
